## 📚 Library Management System (CRUD - Java + MySQL)

## 🔹 Overview

This project is a **Library Management System** developed using **Java** and **MySQL**. It is a **command-line (CLI) based application** designed to manage library records efficiently by performing basic **CRUD operations (Create, Read, Update, Delete)**.

The system allows users to interact with the database through a terminal interface, making it simple and lightweight while focusing on core backend logic and database operations.

---

## 🔹 Project Description

The application is built to simulate how real-world library systems manage data such as book details, availability, and updates. It enables users to:

* Add new book records
* View all available books
* Update existing book information
* Delete book records from the database

The project uses **JDBC (Java Database Connectivity)** to establish communication between the Java application and the MySQL database, ensuring smooth and reliable data handling.

---
## 🔹 Technologies Used

* **Programming Language:** Java
* **Database:** MySQL
* **Concepts:**

  * JDBC
  * Object-Oriented Programming (OOP)
  * SQL Queries
  * CRUD Operations
